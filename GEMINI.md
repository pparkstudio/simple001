# 로또 번호 생성기 - GEMINI.md

이 프로젝트는 HTML, CSS, Vanilla JavaScript로 제작된 간단한 웹 기반 로또 번호 생성기입니다.

## 기술 스택
- **프론트엔드**: HTML5, CSS3, Vanilla JavaScript (ES6+).
- **스타일링**: 표준 CSS (프레임워크 미사용).
- **환경**: `.idx` 설정을 통해 관리됨.

## 프로젝트 구조
- `index.html`: 애플리케이션의 구조 및 진입점.
- `main.js`: 로또 번호 생성 및 표시 로직.
- `style.css`: 시각적 디자인 및 레이아웃.

## 개발 가이드라인
- **유지보수**: 로직은 `main.js`, 스타일은 `style.css`에서 관리합니다.
- **Vanilla JS**: 복잡한 기능이 필요한 경우가 아니라면 외부 라이브러리 사용을 지양합니다.
- **일관성**: `main.js`의 `getNumberColor`에 정의된 번호별 색상 규칙을 준수합니다.
- **스타일링**: 새로운 UI 컴포넌트 추가 시 현재의 디자인(중앙 정렬, 둥근 모서리, 그림자 효과)을 유지합니다.
- **언어**: 사용자 인터페이스와 문서는 한국어를 기본으로 합니다.

## 향후 개선 사항
- 번호 생성 시 애니메이션 효과 추가.
- 생성된 번호 이력(History) 저장 기능.
- 다양한 로또 방식 지원 (예: 연금복권, 파워볼 등).
